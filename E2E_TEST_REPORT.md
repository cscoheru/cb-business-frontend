## 关键问题分类

### 🔴 P0 - 严重问题 (影响核心功能)

1. **仪表盘页面h1元素缺失**
   - 影响: 所有仪表盘导航测试失败
   - 文件: `e2e/dashboard-navigation.spec.ts:22`
   - 错误: `element(s) not found` for h1

2. **登录后导航逻辑错误**
   - 影响: 用户登录后停留在登录页面
   - 文件: `e2e/auth-flow.spec.ts:68`
   - 错误: Expected `/(dashboard|home)/`, received `/login`

3. **注册表单功能不完整**
   - 影响: 无法完成用户注册流程
   - 文件: `e2e/membership.spec.ts`

### 🟡 P1 - 重要问题 (影响用户体验)

4. **升级提示和弹窗未显示**
   - 影响: 免费用户看不到升级引导
   - 文件: `e2e/subscription-flow.spec.ts`, `e2e/membership.spec.ts`

5. **导航链接数量不足**
   - 影响: 桌面端导航栏显示不完整
   - 文件: `e2e/dashboard-navigation.spec.ts:243`
   - 错误: Expected >= 4 links, received 0

6. **订阅状态信息未显示**
   - 影响: 用户无法查看当前订阅状态
   - 文件: `e2e/subscription-flow.spec.ts`

### 🟢 P2 - 次要问题 (边缘情况)

7. **离线模式测试失败**
   - 影响: 网络错误处理验证
   - 文件: `e2e/dashboard-navigation.spec.ts:291`
   - 错误: `net::ERR_INTERNET_DISCONNECTED`

8. **响应式布局问题**
   - 影响: 移动端用户体验
   - 文件: `e2e/dashboard-navigation.spec.ts`

9. **图表组件缺失**
   - 影响: 数据可视化展示
   - 文件: `e2e/dashboard-navigation.spec.ts`
